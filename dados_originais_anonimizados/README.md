Pasta destinada ao armazenamento dos conjuntos de dados anonimizados, gerados a partir dos 30 bancos de dados originais, após a aplicação das técnicas de anonimização consideradas neste estudo (k-anonymity, l-diversity e t-closeness). Como essas técnicas requerem a codificação prévia dos atributos categóricos por meio de label encoding e one-hot encoding, este diretório contém, ao todo, 90 conjuntos de dados.
