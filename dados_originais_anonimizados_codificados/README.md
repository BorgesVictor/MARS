Pasta destinada ao armazenamento dos conjuntos de dados anonimizados, gerados a partir dos 30 bancos de dados originais, após a aplicação das quatro técnicas de anonimização consideradas neste estudo k-anonymity, l-diversity, t-closeness e differential privacy. Para as técnicas k-anonymity, l-diversity e t-closeness, foi necessária a codificação prévia dos atributos categóricos por meio de label encoding e one-hot encoding. Em função disso, este diretório contém, ao todo, 120 conjuntos de dados anonimizados, incluindo versões codificadas e não codificadas, conforme a técnica aplicada.
