Nesta pasta estão armazenados os conjuntos de dados resultantes da aplicação do procedimento de bagging e das técnicas de anonimização consideradas neste estudo, totalizando 1.050 conjuntos de dados. Desse total, 210 correspondem às versões originais geradas a partir do bagging aplicado aos 30 bancos de dados iniciais, enquanto os 840 restantes resultam da aplicação das quatro técnicas de anonimização a esses 210 conjuntos.
